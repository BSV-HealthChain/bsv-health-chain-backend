## Features

- **User Management**: Register and authenticate patients and healthcare providers.
- **Patient Records**: Securely store and retrieve patient health data.
- **Health Providers**: Manage appointments between patients and health providers.
- **Blockchain Integration**: Record healthcare transactions on BSV blockchain for transparency and immutability.


---

## Tech Stack

- **Backend Framework**: Node.js, Express
- **Database**: MongoDB
- **Blockchain SDK**: BSV SDK
- **Authentication**: JWT
- **API Documentation**: Swagger
- **Testing**: Jest / Supertest (optional)
